About Funi PHP

Funi PHP is an open-source PHP web framework that allow developers build web applications quickly and easily using the model–view–controller (MVC) design pattern. It provides rich set of libaries that handles commonly required task allowing the developer focus on custom coding for his/her project

Funi-PHP is just the right framework for you if:

  1. You want a lightweight PHP Framework
  2. You want a framework that is easy to use without a steep learning curve
  3. You want a framework where there is not hassle with setup.
  4. You want a framework that provides valuable libraries to help with complex operations but also give you the fredoom to fully express your coding abilities
  5. You want a framework that is organized, with a simple and straight forward file structure
  6. You need a framework that you can HACK and improve

Installation Instructions

Installing Funi-PHP is as easy as ABC. It involves 3 simple steps:

  1. Open the framework on your browser. Your will be redirected to the installation page
  2. Make sure your computer meet the framework's requirement in the pre-installation check
  3. Enter your Hostname, Database Name,Database Username,Database Password and Timezone in the appropriate form fields. This will help the framework create a local database for you using these parameters
  4. Enter the name of your application. Note Funi PHP folder will be renamed to your application name. Also the baseUrl of your application will be automatically built in the format http://$_SERVER[HTTP_HOST]/appName/public/
  5. Viola...your done. you can now have fun building your application

Deployment

Deploying your application on a live server has never been easier. Funi-PHP provides a deployment interface where you can enter your live server parameters. With this feature in place, you dont have to go to your code to change your parameters whenever you want to go live. Funi-PHP automatically knows your development environment and uses the appropriate parameters to set up your application

To enter your deployment settings :

  1. goto http://$_SERVER[HTTP_HOST]/appName/deployment
  2. Enter your Hostname, Database Name,Database Username,Database Password, App Name and baseUrl in the appropraite form fields. This will help the framework create a local database for you using these parameters Note BaseUrl is not automatically built in the deployment interface. If you want to host your application with a domain name myexampleapp.com, your baseUrl will be http://myexampleapp.com/public/
  3. you are done, you can deploy your application online without any hassle
